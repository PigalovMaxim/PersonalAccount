# Версия node v16.14.2
# Как запускать приложение
- Скачать приложение (Далее работа в корневой папке)
- Написать npm install
- Написать npm start
- Написать node server.js
# Стек технологий, которые использовались: 
- React (+ React Router Dom)
- TypeScript
- Sass
- Redux Toolkit
- NodeJS (+ Express)
- Axios
